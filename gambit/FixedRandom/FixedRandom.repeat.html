Returns a random instance that repeats the given values indefinitely.

Example

final random = FixedRandom.repeat([0.1, 0.2, 0.3]);
print(random.nextDouble()); // 0.1
print(random.nextDouble()); // 0.2
print(random.nextDouble()); // 0.3
print(random.nextDouble()); // 0.1
