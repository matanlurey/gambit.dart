Dice

Dice class final

A single die with a fixed number of sides.

See also:

  • d4
  • d6
  • d8
  • d10
  • d12
  • d20
  • d100

Equality

Two dice are equal if they have the same number of sides:

print(Dice(6) == Dice(6)); // true
print(Dice(6) == Dice(20)); // false

Example

final d6 = Dice(6);
print(d6.sample(Random()));
